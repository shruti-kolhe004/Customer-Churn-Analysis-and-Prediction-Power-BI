# Customer-Churn-Analysis-and-Prediction-Power-BI
# 📊 Overview

This project analyzes customer churn in a telecom company using Power BI. It focuses on identifying patterns and key factors contributing to customer churn.


# 🎯Objectives

*Analyze churn rate

*Identify high-risk customers

*Provide actionable insights


# 📁 Dataset

*1000+ customer records

*Features include: tenure, contract type, payment method, internet service, etc.


# 📊 Dashboards


1. Churn Overview

*Overall churn rate

*Customer distribution


2. Customer Demographics

*Gender, partner, dependents

*Churn comparison


3. Tenure Analysis
   
*Churn higher in early-stage customers


4. Churn Analysis

*High churn in month-to-month contracts

*Payment method and service impact


# 💡 Key Insights

*Customers with short tenure churn more


*Month-to-month contracts have highest churn


*Certain payment methods show higher churn



# 🛠 Tools Used

*Power BI

*DAX

*Data Visualization
